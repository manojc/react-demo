# react-demo
This demo covers basics of `react`.

It uses `typescript` & `webpack` for the compilation.

Follow following steps to setup the project - 

```bash

# install node dependencies
npm i

# build webpack
npm run build

# in case of watch mode
npm start

# server the `index.html`
npm run serve

```
